SW[9:0]   - 10 переключателей
HEX[3:0]  - 4 семисегментых индикатора

Примечания:
- Константа - это такое число, которое выберет преподаватель при сдаче ЛР.
- Результаты вычислений выводятся на семисегментные индикаторы с помощью модуля, написанного в ЛР1. То есть
    RD1[15:12]  на HEX[3]
    RD1[11:8]   на HEX[2]
    RD1[7:4]    на HEX[1]
    RD1[3:0]    на HEX[0]
  и, соответсвенно, если RD1[15:0] = 16'hBEDA, тогда на семисегментных индикаторах загорятся соответствующие
  шестнадцатеричные цифры
    HEX[3] - B
    HEX[2] - E
    HEX[1] - D
    HEX[0] - A

20 вариантов - по списку в группе
1) Вычислить a*b с использованием операций сложений и сдвига (как если бы умножали двоичные числа в столбик)
    a = const, b = SW[7:0] (a и b неотрицательные)
2) n-ый член арифметической прогрессии an
    a1 = const, d = SW[9:6], n = SW[5:0] (d и n неотрицательные)
3) Вычислить n!, где n = SW[2:0]
4) Вычислить циклический сдвиг влево от a = const. Величину сдвига определяет SW[9:0].
    Пример: a = 32'h0000_000f, после такого сдвига на 30 разрядов получим 32'hc000_0003
5) Найти остаток от деления SW[9:0] на a = const
6) Найти количество единиц в двоичном представлении числа SW[9:0]
    Пример: SW = 10'b01_0001_0110, результат вычислений 4
7) Сформировать число, состоящее из четных двоичных разрядов числа SW[9:0]
    Пример: SW = 10'b11_1011_1000, результат вычислений 5'b10100
8) Найти наибольший двоичный разряд числа SW[9:0], значение которого равно 1. Если такого нет, вывести 10
    Пример: SW = 10'b01_0001_0110, результат вычислений 8
9) Вычислить a-b без использования операции вычитания. a = const, b = SW[7:0] (b - знаковое)
10) Поменять в числе a = const местами двоичные разряды i = SW[2:0] и j = SW[5:3]
    Пример: a = 32'h00_00_00_31, i = 3'd0, j = 3'd6, результат вычислений 32'h00_00_00_70
11) Вычислить 2^(SW[3:0])
    Пример: SW = 4'd10, результат вычислений 1024
12) n-ый член последовательности Фибоначчи Fn
    n = SW[3:0]
13) Сформировать число, i-ый двоичный разряд которого равен SW[9-i], где 0 <= i <= 9
    Пример: SW = 10b'11010_11001, результат вычислений 10'b10011_01011
14) Найти наименьший двоичный разряд числа SW[9:0], значение которого равно 0. Если такого нет, вывести 10
    Пример: SW = 10'b01_0001_0110, результат вычислений 0
15) Проверить, является ли число SW[9:0] степенью двойки. Если да, то вывести 1, иначе 0
16) Вычислить n-ый член сиракузской последовательности. Если n-ый член последовательности f(n)
    четный, то f(n+1) = f(n)/2, иначе f(n+1) = f(n) * 3 + 1
    n = SW[9:0], f1 = const > 0
17) Найти количество двоичных разрядов, в которых отличаются числа SW[9:5] и SW[4:0]
    Пример: SW[9:5] = 5'b0_1100, SW[4:0] = 5'b0_1010, результат вычислений 2
18) Определить, сколько раз встречается 2'b11 в двоичном представлении SW[9:0]
    Пример: SW = 10'b11_0101_1110, результат вычислений 4
19) Вычеркнуть i-ый двоичный разряд из a = const (младшие i-го биты остаются на месте, старшие сдвигаются на один разряд вправо). i = SW[2:0]
    Пример: a = 32'h00_00_00_EC, i = 3'd4, результат вычислений 32'h00_00_00_7C
20) Загрузить массив констант из 16 элементов в регистровый файл (преподаватель может изменять значения элементов, но не их количество). Найти среднее арифметическое
